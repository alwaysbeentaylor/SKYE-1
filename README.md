# SKYE - Familie Verbinding App

SKYE is een veilige familie communicatie app met een unieke "Bubble" interface.

## Lokale Development

```bash
# Install dependencies
npm install

# Start backend (Terminal 1)
npm run server

# Start frontend (Terminal 2)
npm run dev
```

## Environment Setup

Kopieer `.env.example` naar `.env` en vul je Firebase credentials in.
